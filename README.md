# Ratp OpenData

A Ruby Client to consume [RATP Opendata](https://data.ratp.fr/).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ratp'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ratp
